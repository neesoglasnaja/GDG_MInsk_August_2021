  • CSS Counter Styles

  • Broadcast Channel API и Channel Messaging API
  • Streams API

  • WebRTC API: Media Stream и MediaStream Recording API
  • Media Capabilities API, Media Source API, Media Session API и Encrypted Media Extensions API

Управление устройствами

  • Geolocation API
  • Permissions API

  • Web Authentication API и Credential Management API
